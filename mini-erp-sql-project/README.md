# Mini ERP SQL Project

This project demonstrates relational database design and ERP-style reporting using MySQL 8.0 (WSL Ubuntu).

## Features
- Normalized database schema
- Primary & Foreign Keys
- CHECK constraints
- JOIN queries
- GROUP BY and aggregation
- Salary reporting
- Department-wise analysis

## Skills Demonstrated
- SQL (Intermediate)
- Relational Database Design
- ERP-style Reporting Queries
- Data Integrity Enforcement
- MySQL CLI (Ubuntu WSL)
